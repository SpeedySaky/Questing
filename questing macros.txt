Install addon from Macro.zip into your wotlk classic addon folder as any other addon you install

\World of Warcraft\_classic_\Interface\AddOns

Log into game and check addon list for addon Macro and enable it.

If addon not on list,then repeat from start,you did not install it right.

When ingame on char,you can check if addon loaded right by opening your macros window and check for macros it should create

Macros by name that you should have are

Hearthstone
Rhapsody
ThunderAle
IF
Thelsamar
